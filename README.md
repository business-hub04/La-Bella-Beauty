# La-Bella-Beauty
La Bella Beauty | Professional Makeup Portfolio & Booking System
La Bella Beauty (LBB) is a high-end, minimalist web application designed for professional makeup artists. It features a seamless client booking experience, a secure owner dashboard for appointment management, and an editorial-style gallery with integrated client testimonials.

Client Booking System (book.html)
Service Selection: Categorized services (General vs. Bridal) with dynamic pricing.

Smart Calendar: Integrated 2026 calendar that tracks availability.

Double-Booking Prevention: Time slots (09:00 - 17:00) become "Taken" once a client has stored a booking for that specific day.

WhatsApp Integration: Automatically generates a formatted message to the artist with the client's details and unique booking ID.

Owner Dashboard (dashboard.html)
Daily Analytics: A "Total Expected" counter that calculates revenue for the current day.

Payment Tracking: Color-coded status badges (Paid in Green, Pending in Red) that the owner can toggle with a click.

Client Management: View client names, surnames, phone numbers (with click-to-chat WhatsApp links), and service details.

Wall of Fame & Gallery (gallery.html)
Interactive Portfolio: Masonry-style grid with a zoom-on-hover effect that displays the makeup style.

Review System: A star-rating and feedback module that captures the date and time of every client testimonial.
